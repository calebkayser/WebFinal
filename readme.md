This website is a simple final project I made for a pretend company called GreenTech Solutions, and it shows how clean technology can help the environment. The site has five pages that explain what the company does, show its products, share helpful resources, and let users contact the team. I built the website using HTML for the structure, CSS for the design, and JavaScript for things like forms, filters, and high‑contrast mode. The site works on phones, tablets, and computers, and I published it online using GitHub Pages. This project helped me practice building a full website from scratch and learn how to upload and host it on GitHub.
